# TELEGRAM BOT BOILERPLATE

## Prerequisites
1. Docker

## Installation

- Copy file **_env.sample.yml_** to file **_env.yml_**. Fill information of env file.

    - Run command line
    > #### Local Environment
    > yarn
    >
    > yarn start:dev

    > #### Deploy Environment
    > docker compose up

- Open your browser and navigate to **http://localhost:3000/api** (port is setup in env file)
